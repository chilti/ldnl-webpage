# Instrucciones para crear og-image.png

## Especificaciones
- **Tamaño**: 1200x630 píxeles
- **Formato**: PNG
- **Ubicación**: public/og-image.png

## Diseño Sugerido

### Colores
- Fondo: Gradiente de #003D7A (azul UNAM) a #6B46C1 (púrpura)
- Texto principal: Blanco (#FFFFFF)
- Acento: #FFB81C (dorado UNAM)

### Contenido
1. **Título Principal** (grande, centrado):
   "Laboratorio de Dinámica No Lineal"
   
2. **Subtítulo** (más pequeño):
   "Universidad Nacional Autónoma de México"
   
3. **Elementos visuales**:
   - Logo UNAM en esquina superior izquierda
   - Visualización abstracta de sistemas dinámicos o redes neuronales en el fondo
   - Líneas o patrones que sugieran complejidad y dinamismo

### Tipografía
- Fuente principal: Inter, Roboto o similar (sans-serif moderna)
- Peso: Bold para título, Regular para subtítulo

## Herramientas Sugeridas
- Canva (plantilla Open Graph)
- Figma
- Adobe Photoshop/Illustrator
- GIMP (gratuito)

## Alternativa Temporal
Mientras tanto, puedes usar una imagen simple con:
- Fondo azul sólido (#003D7A)
- Texto blanco centrado con el nombre del laboratorio
