### Áreas de Investigación

#### Análisis Bibliométrico

Aplicación de técnicas cuantitativas para evaluar la producción científica:

- Análisis de citas
- Mapas de colaboración
- Identificación de tendencias
- Evaluación de impacto

#### Machine Learning

Uso de aprendizaje automático para:

- Clasificación de publicaciones
- Predicción de impacto
- Detección de comunidades científicas
- Análisis de texto

#### Redes Complejas

Estudio de redes de colaboración científica:

- Análisis de redes sociales
- Detección de comunidades
- Centralidad y prestigio
- Evolución temporal

### Software y Herramientas

Hemos desarrollado herramientas para análisis de datos científicos:

- **DataSOMining**: Herramienta para minería de datos con mapas auto-organizados
- **LabSOM**: Laboratorio virtual para análisis con SOM
