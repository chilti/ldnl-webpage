## Proyectos de Docencia

El laboratorio desarrolla proyectos enfocados en mejorar la enseñanza de matemáticas, física y ciencias de la computación.

### Desarrollo de Software Educativo

#### INTEGRA - Software de Sistemas Dinámicos

**Descripción**: Desarrollo de software para la enseñanza de sistemas dinámicos.

**Impacto**:
- Usado en más de 20 universidades
- Más de 1,000 estudiantes beneficiados
- Material didáctico incluido

#### LabSOM - Laboratorio Virtual

**Descripción**: Plataforma educativa para aprendizaje de mapas auto-organizados.

**Características**:
- Experimentos guiados
- Visualización interactiva
- Ejercicios prácticos
- Material de apoyo

### Material Didáctico

#### Apuntes de Cursos

Desarrollo de apuntes completos para:
- Sistemas Dinámicos No Lineales
- Introducción a la Bioinformática
- Ecuaciones Diferenciales con Aplicaciones

**Disponibilidad**: Material de libre acceso para estudiantes

#### Tutoriales en Línea

Creación de tutoriales sobre:
- Python para científicos
- MATLAB para sistemas dinámicos
- Git para investigadores
- Visualización de datos

### Talleres y Cursos

#### Programa de Talleres

Talleres regulares sobre:
- Sistemas dinámicos
- Bioinformática
- Machine learning con SOM
- Análisis de series temporales

**Participantes**: Más de 500 estudiantes en los últimos 5 años

#### Cursos de Verano

Cursos intensivos durante el verano:
- Introducción a la teoría del caos
- Bioinformática computacional
- Ciencia de datos con Python

### Formación de Recursos Humanos

#### Tesis Dirigidas

- **Doctorado**: 5+ tesis
- **Maestría**: 15+ tesis
- **Licenciatura**: 30+ tesis

#### Estancias de Investigación

Programa de estancias para estudiantes de licenciatura y posgrado.

**Duración**: 3-6 meses

**Actividades**:
- Participación en proyectos de investigación
- Desarrollo de software
- Análisis de datos
- Redacción de artículos
