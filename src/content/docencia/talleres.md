## Talleres y Cursos

El Laboratorio de Dinámica No Lineal imparte regularmente talleres y cursos sobre diversos temas relacionados con nuestras áreas de investigación.

### Talleres Regulares

#### Taller de Sistemas Dinámicos

**Descripción**: Introducción a la teoría de sistemas dinámicos no lineales, con énfasis en aplicaciones computacionales.

**Contenido**:
- Ecuaciones diferenciales ordinarias
- Retratos de fase
- Análisis de estabilidad
- Bifurcaciones
- Caos y atractores extraños

**Herramientas**: INTEGRA, MATLAB, Python

**Duración**: 40 horas

**Nivel**: Licenciatura/Posgrado

#### Taller de Bioinformática

**Descripción**: Métodos computacionales para el análisis de secuencias biológicas.

**Contenido**:
- Bases de datos biológicas
- Alineamiento de secuencias
- Búsqueda de similitud (BLAST)
- Análisis filogenético
- Predicción de estructura

**Herramientas**: Python, BioPython, herramientas web

**Duración**: 30 horas

**Nivel**: Licenciatura/Posgrado

#### Taller de Machine Learning con SOM

**Descripción**: Introducción a mapas auto-organizados y sus aplicaciones.

**Contenido**:
- Fundamentos de redes neuronales
- Algoritmo SOM
- Visualización de datos
- Clustering
- Aplicaciones en ciencia de datos

**Herramientas**: LabSOM, DataSOMining, Python

**Duración**: 20 horas

**Nivel**: Posgrado

#### Taller de Análisis de Series Temporales

**Descripción**: Métodos para el análisis de señales y series temporales.

**Contenido**:
- Análisis espectral
- Filtrado de señales
- Análisis no lineal
- Detección de eventos
- Aplicaciones en neurociencia

**Herramientas**: Bansy 3, MATLAB, Python

**Duración**: 25 horas

**Nivel**: Posgrado

### Cursos Semestrales

#### Sistemas Dinámicos (Licenciatura)

Curso semestral sobre teoría de sistemas dinámicos para estudiantes de matemáticas y física.

**Temas**:
- Ecuaciones diferenciales
- Sistemas lineales y no lineales
- Estabilidad
- Bifurcaciones
- Introducción al caos

#### Ecuaciones Diferenciales (Licenciatura)

Curso básico de ecuaciones diferenciales con aplicaciones.

#### Modelación Matemática (Posgrado)

Curso avanzado sobre modelación de sistemas complejos.

### Talleres Especiales
